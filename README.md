# 虫娘ローグガーデン PWA v17

卵から虫娘を孵化させ、朝露の森を探索するPWAブラウザゲームです。

## 今回の主な更新

- 通常敵4体・強敵3体のドット絵画像を追加
- 敵AIを追加
  - 初手は虫娘の一番高い攻撃ステータスに勝つ手を出しやすい
  - 虫娘が勝った後、敵はその勝った手に勝つ手を出しやすい
  - 敵ごとに隠しIQを設定し、高IQほど読み外しを混ぜる
- 戦闘報酬を調整
  - 通常敵は素材メイン＋低確率エサ
  - 強敵は素材多め＋エサ出やすめ＋形見抽選
- 素材訓練を調整
  - 蜜：HP
  - 樹液：防御
  - 花粉：✊✌️✋攻撃
  - 訓練上限と段階式コストを設定
- 探索力を追加
  - 最大3
  - 探索開始時に1消費
  - 20分ごとに1回復
  - オフライン中も回復
- 形見3種・星ランク・銅銀金アイコンは継続実装

## 使い方

GitHubにアップロードする場合は、このフォルダの中身をリポジトリ直下に配置してください。

GitHub Pagesで公開する場合は、Settings → Pages から `main` ブランチの root を公開対象にしてください。

## v20更新：樹液の古木林

- 第2探索地「樹液の古木林」を追加
- 通常敵：樹皮カミキリ、蜜泥ゾウムシ、樹液アリ隊、夜羽ガ
- 強敵：大長刃カミキリ、横綱ハナムグリ
- 夜羽ガの毒状態異常を追加

- 樹液の古木林は朝露の森より素材量・エサ率・形見機会が高め
